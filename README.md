# Blender Product Showcase Skill

一个可复用的 Codex Skill：根据自然语言需求和参考素材，在 Blender 中创建可编辑的产品三维模型、展示动画与渲染结果，并保留适合录屏的完整建模过程。

## 包含内容

- 通用 Skill 工作流与质量标准
- Blender 产品展示通用提示词 DOCX
- 项目需求规范与提示词模板
- Blender Python 管线模板
- 项目初始化和结果校验脚本

## 安装

在 Codex 中输入：

```text
Install https://github.com/pengxiaoan/blender-product-showcase-skill/tree/main/blender-product-showcase
```

也可以将 `blender-product-showcase` 文件夹复制到：

```text
~/.codex/skills/blender-product-showcase
```

## 使用

```text
Use $blender-product-showcase，根据参考图绘制一个可编辑的 Blender 产品模型，生成展示动画和最终渲染，并保留清晰的全过程供录屏。
```

详细输入字段、验收标准和可复用模板请查看 [`blender-product-showcase/SKILL.md`](blender-product-showcase/SKILL.md)。
